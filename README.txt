# Dibujador de Formas con Turtle

Este es un programa de Python que utiliza el módulo `turtle` para dibujar formas en una ventana gráfica. El programa permite al usuario dibujar líneas, cuadrados, círculos, rectángulos y triángulos.

## Funciones

El programa consta de varias funciones:

- `line(start, end)`: Dibuja una línea desde el punto de inicio hasta el punto final.
- `square(start, end)`: Dibuja un cuadrado desde el punto de inicio hasta el punto final.
- `circle(start, end)`: Dibujar un círculo desde el punto de inicio hasta el punto final.
- `rectangle(start, end)`: Dibuja un rectángulo desde el punto de inicio hasta el punto final.
- `triangle(start, end)`: Dibuja un triángulo desde el punto de inicio hasta el punto final.
- `tap(x, y)`: Almacena el punto de inicio o dibuja la forma.
- `store(key, value)`: Almacena un valor en el estado con una clave específica.

## Uso

Para usar este programa, simplemente ejecútalo en un entorno de Python que tenga los módulos `turtle` y `freegames` instalados. Puedes interactuar con el programa utilizando el mouse y el teclado. Aquí están las teclas que puedes usar:

- 'u': Deshacer la última acción.
- 'K': Cambiar el color del lápiz a negro.
- 'W': Cambiar el color del lápiz a blanco.
- 'G': Cambiar el color del lápiz a verde.
- 'B': Cambiar el color del lápiz a azul.
- 'R': Cambiar el color del lápiz a rojo.
- 'l': Cambiar la forma a línea.
- 's': Cambiar la forma a cuadrado.
- 'c': Cambiar la forma a círculo.
- 'r': Cambiar la forma a rectángulo.
- 't': Cambiar la forma a triángulo.

Diviértete dibujando con Turtle!
